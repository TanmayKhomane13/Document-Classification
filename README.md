# 🧠 Project Overview

**Document Classification** is a beginner-friendly NLP project created to implement the concepts learned in an IBM GenAI course. It focuses on converting raw text into numerical form and using it to classify documents into categories.

The project uses:

- Text preprocessing and cleaning.
- Tokenization and vocabulary creation.
- Dataset handling with TorchText.
- Machine learning and deep learning concepts.
- Data analysis and visualization.

This project is intended for learning, practice, and understanding the workflow of a basic document classification system.
